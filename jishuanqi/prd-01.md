# CustomTkinter 现代化计算器
基于 Python + CustomTkinter 开发桌面版标准计算器，全程使用现代UI组件开发，不引入额外图片、音频资源。代码可直接运行，中文展示正常无乱码，运行流畅无逻辑BUG，支持打包为EXE文件，适配课程实训、练手项目、桌面小工具使用。

## 界面布局
1. 固定窗口尺寸，窗口居中显示，界面刷新无闪烁，支持浅色/深色主题切换
2. 顶部显示程序名称与主题切换按钮，布局简洁规整
3. 上方为显示屏区域，分两行展示历史输入与当前计算结果，字体清晰醒目
4. 下方为按键区域，按功能分区排布：数字键、运算符键、功能键、等号键
5. 整体采用圆角、阴影、渐变样式，遵循现代桌面UI设计风格，按键大小均匀、点击区域合理

## 核心功能与规则
1. 基础运算：支持加、减、乘、除四则运算，兼容小数计算
2. 辅助功能：支持清空全部、回退删除单个字符、正负号切换、百分比计算
3. 运算逻辑：自动处理运算优先级，连续运算逻辑正常，避免重复输入运算符
4. 输入限制：限制最大输入位数，防止界面溢出与数值异常
5. 异常处理：除数为零、非法表达式时弹出中文提示，终止错误计算并重置状态
6. 结果展示：计算完成后实时显示最终结果，保留合理小数位数

## 配套功能
1. 主题切换：一键切换浅色/深色两种界面风格，适配不同使用场景
2. 交互反馈：按键点击有视觉变化反馈，提升操作体验
3. 状态提示：运算异常、输入超限等场景给出文字弹窗提示
4. 操作容错：拦截连续重复输入运算符、开头输入运算符等非法操作
5. 历史记录：显示屏保留上一步运算表达式，便于核对与二次计算
6. 全代码中文注释：界面组件、按键事件、运算逻辑、异常判断均添加详细中文注释
7. 中文渲染正常，无乱码、排版错位问题

## 代码开发规范
1. 采用面向对象思想编码，封装主窗口、显示屏、按键面板、运算逻辑等独立模块
2. 界面渲染、事件监听、数学运算、异常处理代码相互解耦
3. 代码命名规范统一，关键逻辑补充中文注释，结构清晰，便于阅读与二次修改
4. 仅依赖 Python 与 CustomTkinter 库，无额外第三方依赖，兼容性良好
5. 采用 MVC 架构分层设计，视图层、控制层、逻辑层分离，方便后期扩展功能
