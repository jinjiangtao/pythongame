# CustomTkinter 科学计算器 V2.0 迭代提示词
基于 Python + CustomTkinter 开发桌面版**科学计算器**，在标准版计算器基础上迭代升级，全程使用现代UI组件开发，不引入额外图片、音频资源。代码可直接运行，中文展示正常无乱码，运行流畅无逻辑BUG，支持打包为EXE文件，适配课程实训、练手项目、桌面工具使用。

## 界面布局（迭代升级）
1. 固定窗口尺寸，窗口居中显示，界面刷新无闪烁，支持浅色/深色主题一键切换
2. 顶部显示程序名称、版本号与主题切换按钮，布局规整美观
3. 上方为显示屏区域，分两行展示历史运算表达式与当前输入/结果，字体清晰醒目
4. 下方按键区域分为两大模块：标准计算区 + 科学计算区，模块划分清晰
5. 科学按键区域放置三角函数、对数、指数、平方、开方、π、e 等专业计算按钮
6. 整体采用圆角、平滑阴影设计，遵循现代桌面UI风格，按键大小统一、间距合理

## 核心功能与规则（迭代新增+增强）
1. **标准运算**：支持加、减、乘、除四则运算，兼容小数、负数计算
2. **科学计算（新增）**：支持平方、立方、n次方、开平方、开立方、平方根、立方根
3. **高级函数（新增）**：支持 sin、cos、tan 三角函数（角度制）、log、ln、π常数、e常数
4. **辅助功能**：清空全部、回退删字、正负号切换、百分比计算、小数点输入
5. **运算逻辑**：自动处理运算优先级，连续运算正常，避免重复运算符、非法开头
6. **输入限制**：限制最大输入长度，防止界面溢出与数值异常
7. **异常处理**：除数为零、非法表达式、超出范围计算时弹出中文提示并自动重置

## 配套功能（迭代升级）
1. 主题切换：浅色/深色模式一键切换，所有组件自动适配配色
2. 交互反馈：按键点击高亮反馈，操作手感清晰
3. 状态提示：异常计算、输入超限、函数错误均有中文弹窗提示
4. 操作容错：自动拦截重复运算符、非法开头、格式错误等问题
5. 历史记录：显示屏保留上一步完整表达式，便于核对与连续计算
6. 全代码中文注释：界面、按键、运算、函数、异常处理均添加详细注释
7. 中文无乱码、排版无错位、界面自适应主题风格

## 代码开发规范
1. 采用面向对象编码，独立封装窗口、显示区、标准按键、科学函数、计算逻辑
2. 界面渲染、事件监听、数学运算、异常处理、函数计算完全解耦
3. 代码命名规范，分层清晰，便于阅读、修改与扩展功能
4. 仅依赖 Python 内置库与 CustomTkinter，无第三方依赖，兼容性极强
5. 采用 MVC 架构分层：视图层、控制层、逻辑层分离，支持后续继续迭代升级
