# Python Tkinter 五子棋游戏开发提示词
使用Python内置tkinter开发桌面双人五子棋游戏，纯原生库，无第三方依赖，代码可直接运行，中文显示正常无乱码，支持打包为EXE程序，适用于课程作业与实训项目。

## 界面要求
1. 程序窗口居中展示，布局规整，控件排版对齐，字体样式统一
2. 绘制15×15标准棋盘，网格线条清晰，棋子落点贴合交叉点位
3. 黑白棋子样式区分明显，视觉效果美观
4. 界面顶部展示对局状态，实时提示落子方、胜负结果信息
5. 底部设置功能按钮区，包含重新开局、悔棋、退出游戏按钮

## 核心游戏功能
1. 双人交替对局，黑棋为先手，鼠标点击棋盘即可完成落子操作
2. 点位校验，已有棋子位置无法重复落子，拦截无效操作
3. 自动判定胜负，检测横、竖、正反斜线四个方向，达成五子连线判定获胜
4. 棋盘布满棋子且无五子连线，自动判定对局平局
5. 对局结束后锁定落子权限，无法继续下棋

## 拓展操作功能
1. 重新开局：清空棋盘棋子，重置对局状态，恢复初始对局模式
2. 悔棋操作：撤回上一步落子步骤，无棋子可撤回时给出提示
3. 退出游戏：正常关闭程序，运行终止无异常报错
4. 各类操作、对局结果均搭配中文文字弹窗提示

## 交互体验
1. 鼠标点击响应迅速，棋子落点精准无偏移
2. 程序运行流畅，界面无闪烁卡顿，运行稳定不闪退
3. 完善异常容错处理，规避各类操作引发的程序问题

## 代码编写规范
1. 仅使用tkinter内置库开发，不引入额外第三方工具库
2. 功能模块化拆分，棋盘绘制、落子逻辑、胜负判断、事件触发各自封装独立函数
3. 代码附带详细中文注释，核心算法逻辑标注说明
4. 代码逻辑严谨无漏洞，复制后可直接运行调试
5. 程序结构规范，预留优化拓展空间
6. 界面采用MVC 架构组织代码， 界面负责展示，逻辑负责处理，数据负责存储，每个模块单独的目录。
7. 界面提示文字都用中文字符， 保证中文显示正常无乱码。
