# CustomTkinter 力学原理科普小游戏
基于 Python + CustomTkinter 开发面向小学生的力学启蒙互动游戏，全程使用原生UI组件，不引入外部图片、音频资源。代码可直接运行，中文显示正常无乱码，运行流畅无逻辑BUG，支持打包为EXE，适用于小学科学课堂教学、家庭趣味科普、课后练习使用。

## 界面布局
1. 窗口固定尺寸，启动自动居中，配色清新护眼，风格童趣且贴合科普场景
2. 顶部区域：游戏标题、当前实验关卡、知识点名称、操作提示、主题切换按钮
3. 中部核心区：**物理模拟画布**，展示小球、滑块、斜面、杠杆、弹簧等模拟对象，实时呈现运动状态与受力效果
4. 下方控制区：功能操作按钮、参数调节滑块、重置按钮、下一关按钮、答案/知识点查看按钮
5. 底部状态栏：实时运动数据（速度、距离、状态）、知识点简要解读
6. 整体采用圆角控件、标准字体，元素布局疏密合理，操作按钮大小适配儿童操作，支持浅色/护眼深色主题一键切换
7. 画布区域视野充足，运动轨迹、物体形态展示完整，无遮挡、无界面闪烁

## 核心功能与规则
1. 分关卡设计力学实验场景，循序渐进讲解基础力学定律，覆盖**重力、摩擦力、斜面受力、杠杆原理、弹力、惯性**六大基础知识点
2. 交互式模拟玩法：用户可拖动物体、调节参数（斜面角度、接触面粗糙程度、配重大小、拉力强弱），实时观察物体运动变化
3. 场景规则：
   - 重力关卡：释放不同高度物体，观察下落速度、落地时间差异，理解重力作用
   - 摩擦力关卡：切换光滑/粗糙接触面，对比滑块滑行距离，认知摩擦力影响
   - 斜面关卡：调节斜面倾角，观察物体下滑快慢，理解斜面省力原理
   - 杠杆关卡：移动支点、增减两端配重，观察杠杆平衡状态，掌握杠杆平衡条件
   - 弹力关卡：拉伸/压缩弹簧，观察形变与回弹效果，认识弹力特性
   - 惯性关卡：快速抽走支撑物，观察上方物体保持原有状态，理解惯性现象
4. 互动答题机制：每轮模拟结束后弹出随堂小问题，结合观察到的现象作答，校验学习效果
5. 参数可视化：滑块调节数值实时显示，运动轨迹可选择是否绘制，方便观察规律
6. 状态实时反馈：物体运动、静止、形变等状态同步文字说明，辅助理解现象

## 配套功能
1. 分步引导提示：新手进入关卡时给出简易操作指引，降低上手难度
2. 知识点解读：一键查看对应力学原理通俗讲解，语言贴合小学生认知，避免专业晦涩术语
3. 一键重置功能：快速还原实验初始状态，支持反复多次模拟测试
4. 答题反馈：答对给予鼓励文案，答错结合实验现象给出引导提示，不直接公布答案
5. 进度记录：自动记录已完成关卡、答题正确率，直观展示学习进度
6. 运动数据展示：实时显示下落时间、滑行距离、倾斜角度等参考数值，量化物理现象
7. 全代码添加中文注释，界面组件、物理模拟逻辑、交互事件、数据计算注释完整清晰

## 代码开发规范
1. 采用面向对象编码，独立封装主窗口、物理画布渲染、各场景模拟逻辑、参数控制、答题校验、关卡管理模块
2. 界面渲染、交互事件、物理运算、状态反馈代码完全解耦，单个功能独立维护
3. 代码命名规范统一，结构层级清晰，便于调试、修改以及后续新增力学场景
4. 仅依赖 Python 与 CustomTkinter 库，无额外第三方依赖，兼容性强，可正常打包为EXE程序
5. 采用 MVC 架构分层，视图层、控制层、物理逻辑层分离，扩展性强，可继续拓展浮力、滑轮等更多力学场景
6. 代码按功能拆分多个文件，禁止所有代码写在同一文件内，方便维护迭代
7. 全界面使用中文展示，无英文内容，适配小学生使用习惯
8. 物理运算逻辑简化适配教学场景，模拟效果贴合基础物理规律，兼顾趣味性与科学性。
