# Python 员工考勤管理系统 V1.0 基础版 
基于 Python + Tkinter 开发桌面版员工考勤管理系统V1.0基础版，全程原生GUI组件开发，不使用外部图片、复杂第三方UI资源。代码可直接运行，中文展示正常无乱码，运行流畅无卡顿、无逻辑BUG，支持打包为EXE文件，适配课程设计、实训项目、期末作业使用。

## 界面布局
1. 固定窗口尺寸，窗口居中显示，比例协调美观
2. 顶部菜单栏：系统、帮助
3. 左侧功能按钮区：员工管理、考勤打卡、记录查询、退出系统
4. 中间主区域：表格展示数据，支持滚动查看
5. 底部状态栏：显示当前操作提示、运行状态
6. 整体商务简洁风格，布局规整、操作直观

## 核心功能（V1.0）
1. 员工信息管理：新增、查询、展示员工列表（工号、姓名、部门）
2. 考勤打卡：根据工号快速签到/签退，自动记录时间
3. 考勤记录查询：展示全部打卡记录，支持按工号筛选
4. 基础状态：正常、迟到、早退、缺勤
5. 数据本地存储，重启不丢失
6. 基础操作弹窗提示：成功/失败/警告

## 配套功能
1. 表格实时刷新，数据展示清晰
2. 空值校验、重复打卡校验
3. 操作日志状态栏提示
4. 全中文注释，无乱码
5. 仅使用Python内置库，开箱可运行

## 代码开发规范
1. 面向对象编码，模块拆分清晰
2. 界面、逻辑、数据分离
3. MVC架构基础版
4. 代码简洁、易读、易修改
