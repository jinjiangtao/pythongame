现在我需要你帮我用 Python 开发一个简易画图板工具，界面友好，功能实用。要求如下：
一、核心绘图功能
1. 画笔工具：鼠标左键拖拽自由绘制，可调节画笔粗细（1-20像素）和画笔颜色
2. 橡皮擦：擦除绘制内容，大小可调，本质是绘制背景色的画笔
3. 形状绘制：
   · 直线：鼠标按下到松开绘制直线
   · 矩形：拖拽绘制空心或实心矩形
   · 圆形/椭圆：拖拽绘制空心或实心圆/椭圆
4. 填充工具：点击封闭区域，用当前颜色填充（泛洪填充算法）
5. 颜色选择：
   · 预设颜色按钮（黑、白、红、绿、蓝、黄等）
   · 自定义颜色选择器（调用系统颜色对话框）

二、画布操作
6. 新建画布：清空当前画布，提示保存（如有未保存内容）
7. 保存画布：保存为 PNG/JPG 格式，使用文件对话框选择路径
8. 打开图片：打开本地图片（PNG/JPG/BMP）到画布上继续编辑
9. 撤销/重做：至少支持 20 步撤销，可重做
10. 清空画布：一键清空所有绘制内容
三、界面布局
· 窗口大小 1000x700
· 左侧工具栏（宽150）：工具按钮（画笔、橡皮擦、直线、矩形、圆、填充）、颜色选择区、粗细滑动条
· 右侧画布区（宽850）：白色绘图区域，带滚动条（画布可大于窗口）
· 顶部菜单栏：文件（新建、打开、保存、退出）、编辑（撤销、重做、清空）、帮助（关于）
四、技术要求
· 使用 CustomTkinter 
· 画布使用 tkinter.Canvas 
· 填充算法用队列实现泛洪填充（注意防止递归栈溢出）
· 撤销功能使用命令模式或保存历史快照（每步保存画布状态）
· 橡皮擦和画笔性能优化：鼠标移动时绘制连续点，避免断线
· 支持鼠标坐标实时显示（状态栏）
五、文件结构
```
drawing_app/
├── main.py          # 程序入口
├── app.py           # 主窗口类
├── canvas.py        # 画布管理（绘图逻辑）
├── tools.py         # 工具类（画笔/形状/填充）
├── history.py       # 撤销/重做管理
└── utils.py         # 辅助函数
```
