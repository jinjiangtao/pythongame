# Pygame 太空陨石躲避战
用 Python Pygame 做一款轻量爽快的太空生存小游戏，采用 MVC 架构分层，分 model / view / controller 目录，代码干净、结构清晰。
全程纯代码绘制，不用任何图片素材，运行流畅、无 bug、可直接打包 exe。

## 游戏玩法
玩家操控一艘小飞船在太空中飞行，屏幕会不断掉落各种大小、速度不一的陨石。
你的目标只有一个：**活下去！坚持越久得分越高。**
飞船可以上下左右移动，不能被陨石砸到，砸到就游戏结束。

## 界面风格
简洁太空风，黑色背景，星星点点，飞船白色，陨石灰色大小不一。
左上角显示生存时间 + 得分，右上角显示生命值。
界面干净、视觉舒服，适合课程展示。

## 核心功能
- 用鼠标流畅控制飞船移动
- 陨石随机大小、随机速度、自动下落
- 陨石砸中飞船扣血，血量归零游戏结束
- 生存时间越长，难度自动提升
- 游戏结束显示最终得分
- 支持重新开始、退出游戏
- 游戏主界面有游戏说明，方便用户玩游戏。

## MVC 架构说明
- model：存放飞船数据、陨石列表、分数、血量、难度
- view：绘制太空背景、星星、飞船、陨石、UI文字
- controller：处理按键、移动逻辑、陨石生成、碰撞判断、难度递增
