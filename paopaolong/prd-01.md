# Pygame 泡泡龙休闲游戏开发提示词
基于 Python + Pygame 开发经典泡泡龙消除类小游戏，全程纯代码绘制图形与界面，不使用外部图片、音频资源。代码可直接运行，中文展示正常，运行流畅无卡顿、无逻辑BUG，支持打包为EXE文件，适配课程作业与实训项目使用。

## 界面布局
1. 固定游戏窗口尺寸，保持稳定帧率，画面刷新无闪烁
2. 顶部状态栏：实时展示得分、当前关卡、游戏状态提示
3. 中间主区域：六边形网格布局彩色泡泡阵列，为核心游玩区域
4. 底部操作区：放置可旋转发射炮台，同时预览下一发泡泡样式
5. 整体配色区分度高，界面元素排布规整，视觉体验舒适

## 核心玩法与规则
1. 鼠标移动控制炮台左右旋转，调整发射瞄准角度，点击左键发射泡泡
2. 飞行泡泡触碰墙体可反弹，碰到其他泡泡后固定位置
3. 当三颗及以上同色泡泡相邻时触发消除，相连的悬空泡泡同步脱落并计分
4. 上方泡泡阵列会缓慢整体下移，触达底部区域则游戏结束
5. 单关泡泡全部消除即可通关，自动进入下一关卡，关卡越高初始泡泡数量越多、下移速度越快

## 配套功能
1. 预判预览：实时显示下一颗待发射泡泡，提升操作策略性
2. 状态提示：对局中、通关、游戏结束均配有文字提示
3. 基础操作：支持重新开局、退出游戏功能
4. 逻辑容错：优化碰撞检测、边界判定，避免角色卡死、元素错位等异常问题
5. 游戏左侧边栏要支持显示当前得分、当前关卡、游戏状态等基本信息
6. 游戏右侧边栏要支持显示当前泡泡样式、预判下一发泡泡样式
7. 游戏开始的时候界面要说明游戏的玩法。
8. 所有游戏元素都要有中文注释，包括界面元素、游戏逻辑、事件处理等。
9. 中文不能乱码

## 代码开发规范
1. 采用面向对象思想编码，合理拆分功能模块，泡泡、炮台、游戏主逻辑各自封装
2. 事件监听、物理运动、碰撞判断、计分规则、画面绘制代码解耦
3. 代码命名规范，关键逻辑添加中文注释，结构清晰便于阅读与二次修改
4. 仅依赖Pygame库，无额外第三方依赖，兼容性良好
5. 架构设计采用MVC 架构，每个模块分开。
