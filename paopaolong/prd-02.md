# Pygame 泡泡龙游戏 V2.0 迭代升级提示词
基于已完成的基础版泡泡龙游戏进行功能迭代，保留原版 MVC 架构、面向对象编码、中文注释与无外部素材特性。迭代后游戏玩法更丰富、体验更完整，代码结构不变、可直接无缝升级，依旧支持直接运行、打包 EXE、课程作业提交。

## 本次迭代目标
在原版核心玩法上增加新机制、新道具、新界面、新难度系统，让游戏从基础版升级为完整可发布版本。

## 界面迭代优化
1. 新增游戏开始菜单界面，显示玩法说明、开始游戏、设置、退出按钮
2. 新增游戏暂停界面，支持继续游戏、重新开始、返回主菜单
3. 新增游戏胜利 / 失败结算界面，显示本局得分、关卡、耗时
4. 左右侧边栏样式美化，增加图标式展示，UI 更精致
5. 泡泡消除、掉落、发射增加基础动画效果，不使用图片

## 核心玩法迭代功能
1. 新增道具泡泡系统
   - 炸弹泡泡：消除周围一圈泡泡
   - 彩虹泡泡：可替代任意颜色泡泡
   - 穿透泡泡：可直接穿过 1 个异色泡泡再固定
2. 新增难度动态变化系统
   - 关卡越高，泡泡下移速度越快
   - 后期随机出现障碍物泡泡，不可消除，增加策略性
3. 新增生命值系统
   - 玩家初始 3 点生命值
   - 泡泡触底不直接结束，而是扣除生命
   - 生命归零游戏结束
4. 新增连击计分系统
   - 连续消除得分翻倍
   - 界面显示连击次数动画

## 体验优化迭代
1. 炮台发射角度增加辅助瞄准线，更易操作
2. 泡泡飞行速度可根据关卡动态调整
3. 增加泡泡消除、掉落、发射的动画效果
4. 游戏内所有提示文字自动居中、颜色区分、更醒目
5. 优化物理碰撞与反弹逻辑，手感更顺滑

## 代码架构迭代要求
1. 完全保留原版 MVC 架构，不破坏原有结构
2. 新增功能模块化封装：道具系统、动画系统、UI系统、生命系统
3. 所有新增代码保持中文注释、命名规范、逻辑清晰
4. 不新增第三方库，仅使用 Pygame 实现所有迭代内容
5. 兼容原版所有功能，可随时回滚，迭代后无 BUG、无卡顿
