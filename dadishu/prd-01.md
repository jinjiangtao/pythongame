# Pygame 打地鼠小游戏
基于 Python + Pygame 开发鼠标操控打地鼠游戏，纯代码绘制图形，不使用外部图片、音频资源。代码可直接运行，运行流畅无逻辑BUG，支持打包为EXE文件，适合趣味互动、反应力训练使用。

## 界面布局
1. 窗口固定尺寸，启动自动居中，绘制多组地洞布局，排布整齐清晰
2. 地洞区域为游戏主画面，地鼠随机从洞口探出、缩回
3. 顶部展示当前得分、剩余游戏时间、游戏状态提示
4. 整体配色简洁明快，元素大小适配鼠标点击操作，视觉无杂乱干扰

## 核心功能与规则
1. 鼠标点击操作，移动光标对准探出的地鼠并单击即可完成击打
2. 地鼠随机从不同洞口出现，停留时长不等后自动缩回洞内
3. 成功击中地鼠累计得分，未及时击打则地鼠自动消失，不得分
4. 设定全局倒计时，时间耗尽本局游戏结束，展示最终得分
5. 地鼠出现位置、出现顺序完全随机，玩法具备不确定性

## 配套功能
1. 游戏结束后支持按键重新开局，可反复游玩
2. 配备退出快捷键，操作简单直观
3. 击打、空击配有对应状态视觉反馈，交互体验完整
4. 全代码添加中文注释，随机生成、鼠标判定、计时计分逻辑说明完整清晰

## 代码开发规范
1. 采用面向对象方式编码，独立封装地洞、地鼠、计分计时、主游戏逻辑等模块
2. 画面渲染、鼠标事件监听、状态切换、数据统计代码相互解耦
3. 代码命名规范统一，结构层次清晰，便于阅读、调试与二次修改
4. 仅依赖 Python 与 Pygame 库，无额外第三方依赖，兼容性良好
5. 代码按功能模块拆分多个文件，禁止单文件整合，方便维护迭代
6. 界面所有提示文字使用中文，上手门槛低。
