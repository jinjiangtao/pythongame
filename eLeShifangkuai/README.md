## 🔧 安装步骤

### 1. 检查Python环境

确保您的系统已安装 Python 3.7 或更高版本：

```bash
python --version
# 或
py --version
```

### 2. 安装 pygame 依赖

```bash
# 使用 pip 安装
pip install pygame

# 或使用国内镜像（更快）
pip install pygame -i https://pypi.tuna.tsinghua.edu.cn/simple

# 或使用 py 命令（Windows）
py -m pip install pygame
```

### 3. 验证安装

```bash
python -c "import pygame; print(pygame.__version__)"
```

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| ← | 向左移动方块 |
| → | 向右移动方块 |
| ↑ | 顺时针旋转方块 |
| ↓ | 加速下落 |
| 空格 | 快速落底（直接落到底部）|
| P | 暂停/继续游戏 |
| Enter | 开始游戏/重新开始 |
| ESC | 退出游戏 |

## 🏗️ 代码架构

游戏代码采用模块化设计，分为以下几个核心模块：

### 1. 常量配置模块 (GameConfig)

## 🎯 游戏技巧

1. **利用预览**：观察下一个方块，提前规划放置位置
2. **优先消除**：尽量消除整行，避免堆积
3. **控制高度**：保持底部区域相对空旷
4. **利用墙壁**：某些方块可以利用墙壁进行特殊操作
5. **快速落底**：用空格键快速落底可以节省时间
