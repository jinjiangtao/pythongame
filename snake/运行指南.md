## 🔧 如果遇到问题

### 问题1：提示 "python 不是内部或外部命令"

**解决方案：**
```powershell
# 使用完整路径运行
& 'C:\Users\84012\AppData\Local\Programs\Python\Python314\python.exe' snake_game.py
```

### 问题2：提示 "No module named 'pygame'"

**解决方案：**
```powershell
# 安装 pygame-ce（推荐）
pip install pygame-ce

# 或者尝试标准版
pip install pygame
```

### 问题3：游戏窗口没有出现

**可能原因：**
- pygame 需要图形显示环境
- 可能在无头服务器环境中运行

**解决方案：**
1. 确保在有图形界面的 Windows 系统中运行
2. 检查是否安装了必要的图形驱动
3. 尝试运行测试脚本验证 pygame 显示功能

### 问题4：终端显示导入错误

**解决方案：**
```powershell
# 运行测试脚本查看详细错误
python test_game.py
```

## 🎮 游戏操作说明

### 控制键
| 按键 | 功能 |
|------|------|
| ↑ 或 W | 向上移动 |
| ↓ 或 S | 向下移动 |
| ← 或 A | 向左移动 |
| → 或 D | 向右移动 |
| 空格键 | 暂停/继续 |
| R | 重新开始（游戏结束后） |
| Q / ESC | 退出游戏 |
| Enter | 开始新游戏 |

### 游戏规则
- 使用方向键或WASD控制蛇的移动方向
- 吃到红色食物得分，蛇身变长
- 撞到边界或自己的身体游戏结束
- 每得50分，速度加快一次
- 尝试打破你的最高分记录！

## 🎯 快速开始

1. **验证环境**（可选）
   ```powershell
   python test_game.py
   ```

2. **启动游戏**
   ```powershell
   python snake_game.py
   ```

3. **开始游戏**
   - 游戏窗口打开后，按 **Enter** 或 **空格键** 开始
   - 使用方向键控制蛇
   - 按 **空格键** 暂停

---

## ⚠ 注意事项

1. **Pygame 窗口**：游戏会打开一个窗口，窗口大小 800x600 像素
2. **关闭游戏**：关闭窗口或按 Q/ESC键退出
3. **最高分**：会自动保存到 `snake_high_score.json` 文件
4. **Python版本**：建议使用 Python 3.8 或更高版本

---

## 🐛 故障排除

如果游戏无法运行：

1. **检查Python**：确保 Python 已正确安装
   ```powershell
   python --version
   ```

2. **检查pygame**：确保 pygame 已安装
   ```powershell
   python -c "import pygame; print(pygame.version.ver)"
   ```

3. **运行测试**：使用测试脚本检查所有依赖
   ```powershell
   python test_game.py
   ```

4. **查看错误**：如果出现错误，运行测试脚本查看详细错误信息
