# CustomTkinter 儿童数字数数益智游戏
基于 Python + CustomTkinter 开发儿童数字数数益智小游戏，全程使用现代UI组件开发，不引入外部图片、音频资源。代码可直接运行，中文显示正常无乱码，运行流畅无BUG，支持打包为EXE文件，适用于低龄儿童数字启蒙、课堂练习与趣味互动。

## 界面布局
1. 窗口固定尺寸，启动自动居中，界面配色柔和护眼，整体风格童趣简洁
2. 顶部区域展示游戏标题、当前关卡、答题正确率、闯关提示
3. 中间为主游戏区，随机展示不同数量的图形/图标，布局宽松、元素大小适中，便于孩子观察点数
4. 下方设置数字选择按钮组、提交按钮、下一题按钮、重新开始按钮，按钮尺寸偏大，适配儿童点击操作
5. 底部状态栏显示操作引导文字、对错反馈提示
6. 整体控件统一圆角样式，字体放大加粗，视觉舒适，无多余杂乱元素
7. 支持浅色、护眼深色主题一键切换

## 核心功能与规则
1. 游戏玩法：界面随机出现若干相同图案，孩子数清数量后点击对应数字作答
2. 题目随机生成，每道题图案样式、数量不重复，数量范围随关卡逐步提升难度
3. 答题校验：点击数字并提交后，自动判断答案对错，给出明确反馈
4. 关卡机制：答对指定题数自动晋级下一关，图案数量、布局复杂度逐步增加
5. 错题处理：答错可重新选择答案，不会直接跳转题目，加深练习记忆
6. 数据统计：自动累计答对题目数、总答题数，计算正确率
7. 限制恶意重复点击，避免误操作影响游戏流程

## 配套功能
1. 全中文操作指引，文字简单易懂，贴合儿童认知
2. 答题后即时弹出对错提示，答对给予鼓励文案，答错给出引导提示
3. 支持手动切换题目、一键重置整局游戏
4. 关卡进度实时展示，清晰知晓当前学习进度
5. 界面元素间距合理，防止点击误触
6. 全代码添加中文注释，界面组件、答题逻辑、数据统计说明完整清晰

## 代码开发规范
1. 采用面向对象方式编码，独立封装主窗口、题目生成、答题校验、关卡管理、数据统计等模块
2. 界面渲染、点击事件、游戏逻辑、状态反馈代码完全解耦
3. 代码命名规范统一，层级结构清晰，便于阅读、调试与二次修改
4. 仅依赖 Python 与 CustomTkinter 库，无额外第三方依赖，兼容性良好
5. 采用 MVC 架构分层设计，视图层、控制层、逻辑层分离，方便后续扩展题型与难度
6. 代码按功能模块拆分，文件不能写到一个文件中，分开写，便于维护。
7. 全程使用中文界面，不出现英文内容，降低使用门槛
8. 界面配色、元素尺寸遵循儿童使用习惯，美观护眼。
