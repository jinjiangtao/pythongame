# 3D装修效果图生成工具

一个简洁高效的在线工具,可以将户型图自动转换为3D装修效果图。

## 🚀 快速开始

### 运行方式

由于项目使用了现代 JavaScript 模块(ES6 Modules),需要通过本地服务器运行:

#### 方法1: 使用 Node.js (推荐)

```bash
# 进入项目目录
cd jszhuangxiu

# 使用 npx 启动简易服务器
npx http-server -p 8000

# 或者使用 Python
python -m http.server 8000

# 然后在浏览器打开: http://localhost:8000
```

#### 方法2: 直接打开(需要浏览器设置)

如果你选择直接打开 `index.html` 文件,可能会遇到模块加载问题。可以尝试:

1. 使用 Chrome 浏览器
2. 右键点击 `index.html` → 打开方式 → Chrome
3. 如果提示 CORS 错误,请使用上面的方法1

## 📖 使用步骤

### 1. 上传户型图
- 点击上传区域或拖拽 JPG/PNG 格式的户型图
- 图片会自动显示在左侧画布上

### 2. 绘制墙体
- **点击画点**: 在图片上点击绘制墙体轮廓的每个顶点
- **双击闭合**: 双击完成当前房间的绘制
- 系统会自动闭合多边形并标注房间编号

### 3. 绘制多个房间
- 至少绘制 **2-3 个房间**
- 每个房间会使用不同颜色区分
- 可以随时点击"撤销"按钮修正

### 4. 生成3D效果
- 点击"生成3D"按钮
- 系统自动生成立体墙壁、地板和家具
- 每个房间中心会自动放置对应类型的家具

### 5. 查看和操作3D效果
- **旋转视角**: 鼠标左键拖拽
- **缩放**: 鼠标滚轮
- **平移**: 鼠标右键拖拽
- 点击"重置场景"可重新开始

## 🛠️ 技术栈

- **HTML5**: 语义化页面结构
- **CSS3**: 现代响应式设计
- **JavaScript (ES6+)**: 原生 JavaScript
- **Three.js**: 强大的 3D 渲染引擎
- **OrbitControls**: 3D 场景交互控制

## 💡 提示

- 使用清晰的户型图可以获得最佳效果
- 绘制时尽量保持点与点之间的均匀分布
- 家具会根据房间自动适配(卧室、客厅、餐厅)
- 可以随时撤销和重新绘制

## 🔧 扩展开发

代码采用模块化设计,方便后续扩展:

- `upload.js`: 可扩展支持更多图片格式
- `wall-drawer.js`: 可添加更多绘图工具
- `scene-3d.js`: 可添加更多家具类型和材质

## 📝 注意事项

- 图片大小限制: 10MB
- 支持格式: JPG, PNG
- 建议至少绘制 2-3 个房间
- 浏览器需要支持 WebGL

## 🎯 核心逻辑

```
用户手动描墙体 
    ↓
程序读取多边形坐标 
    ↓
在 Three.js 里把每段墙生成立方体 
    ↓
地板铺平 
    ↓
放家具
```
