使用 Python + tkinter + sqlite3 开发**简易单机记事本桌面程序**，禁止所有代码写在同一文件，严格模块化分包开发，使用 unittest 编写独立测试文件，结构清晰低耦合，数据持久化存储，可通过 sqlite3 命令行查看数据。

目录结构：
- main.py 程序入口，启动主窗口
- src/
  - main_window.py 主界面、菜单栏、文本编辑区、按钮布局
  - note_core.py 记事本核心逻辑：新建、保存、删除、查询笔记
  - db.py sqlite3数据库模块，实现笔记增删改查
  - config.py 全局配置：窗口大小、数据库名、样式参数
- tests/
  - test_note_core.py
  - test_db.py

功能规则：
1. 纯单机桌面程序，图形化窗口操作；
2. 支持新建笔记、编辑内容、保存笔记、删除笔记、查看所有笔记；
3. 笔记标题+内容存储到sqlite3数据库，重启程序数据不丢失；
4. 可使用 sqlite3 命令行工具打开数据库查看笔记内容；
5. 界面包含文本输入框、保存按钮、删除按钮、清空按钮；
6. 基础异常处理，空内容禁止保存，操作弹窗提示。

开发要求：
UI界面与业务逻辑分离，类职责单一；测试用例可独立运行；代码注释规范，可打包成exe直接运行。
注意：界面中文显示正常， 避免使用中文。
