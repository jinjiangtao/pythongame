使用 Python 内置 tkinter 开发经典桌面学生信息管理系统，纯原生库，无需第三方依赖，代码完整可直接运行。 代码中文显示正常，用户可以中文操作。

## 界面要求
窗口居中、布局整洁、控件对齐、字体统一、排版美观。
使用 Treeview 表格展示学生数据，结构清晰，分区明确。

## 数据字段
包含：学号、姓名、性别、年龄、班级、成绩。
学号唯一，不可重复。

## 用户管理
1. 登录系统，输入用户名和密码。
2. 注册新用户，输入用户名、密码、确认密码。
3. 登出系统，点击登录按钮。

## 学生管理
1. 新增学生信息，做非空判断、数字校验，重复学号拦截
2. 删除选中的学生数据，未选中给予弹窗提示
3. 修改学生信息，点击表格自动回填数据，更新后刷新表格
4. 支持按姓名、学号模糊查询学生信息
5. 一键清空输入框、一键刷新所有数据
6. 所有操作配有弹窗提示（成功、错误、警告）

## 代码要求
1. 代码结构清晰，功能函数分开，注释详细，容错性强，无报错、无崩溃，适合课程作业和实训项目，可直接打包 exe。
2. 代码中文显示正常，用户可以中文操作。
3. 代码中不使用任何第三方库，仅使用 tkinter 库。
4. 代码中不使用任何数据库，仅使用内存中的数据结构。
5. 代码不能写到同一个文件中， 需要分开写。代码采用MVC架构。
6. 代码界面的显示用中文显示，要保证中文不能乱码，正常显示。
