# Pygame弓箭射击小游戏开发提示词
使用Python Pygame开发弓箭射击休闲小游戏，采用MVC分层架构，划分model、view、controller独立目录，职责分离模块化编码。纯代码绘制画面，无额外外部素材，代码完整可直接运行，中文显示正常，运行稳定无报错，支持打包EXE，适用于课程实训作业。

## 界面布局要求
1. 游戏窗口尺寸固定，画面帧率稳定，运行无卡顿闪烁
2. 整体场景风格统一，绘制射击场地、远景背景
3. 界面角落实时展示分数、剩余箭矢、游戏生命值、对局状态
4. 操作提示文字醒目排布，界面配色协调，元素层级分明

## 核心游戏功能
1. 操控射手左右移动，调整射击瞄准角度
2. 按住鼠标蓄力，松开释放弓箭，力度不同飞行轨迹有差异
3. 随机生成移动靶子、飞鸟目标，命中目标累计得分
4. 箭矢触碰边界、地面自动消失，限制射击有效范围
5. 设定箭矢数量上限，耗尽后结束本轮对局
6. 目标脱靶扣除生命值，血量归零游戏结束

## 拓展附加功能
1. 难度递增机制，后续关卡目标移动速度逐步加快
2. 随机刷新加分道具，命中后额外增加箭矢与分数
3. 支持重新开局、暂停游戏、退出游戏操作
4. 命中、脱靶、蓄力阶段配有文字状态提示
5. 完善操作容错判定，规避异常操作引发程序问题

## MVC架构分层规范
1. Model层：存储角色、箭矢、靶子、分数血量等数据模型，处理数据逻辑
2. View层：负责所有画面绘制、界面渲染、文字信息展示
3. Controller层：监听键鼠事件，控制移动射击逻辑、碰撞检测、游戏流程

## 代码编写要求
1. 三层目录独立拆分，代码耦合度低，便于功能迭代修改
2. 功能逻辑封装独立函数，代码命名规范，附带详细中文注释
3. 仅依赖Pygame库开发，不引入其他第三方插件
4. 画面动作流畅自然，碰撞判定精准，无逻辑BUG
5. 整体结构规范，满足作业提交标准，可直接打包发布
6. 中文支持，所有文字均采用中文显示，无英文字符，不能乱码。
