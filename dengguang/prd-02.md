# Pygame 灯光迷阵 V2.0 迭代升级版 
基于原版灯光迷阵进行功能迭代，保留原版所有核心玩法、多文件模块化架构、中文注释、无外部资源。
迭代后游戏更完整、更美观、更有可玩性，代码结构不变，方便维护与扩展。

## 迭代新增内容
1. 主菜单增加：继续游戏、难度选择、主题切换、音效设置
2. 新增限时模式、限步模式，时间/步数用完则游戏失败
3. 新增障碍格子、冰冻格子，增加策略性
4. 新增道具系统：提示、重置、随机点亮
5. 新增关卡星级评价（1~3星）
6. 新增存档系统，记录关卡进度、主题、音效、最高分
7. 新增4套主题：经典、霓虹、糖果、夜间模式
8. 新增点击、翻转、通关、背景音乐，纯代码生成
9. 新增灯格渐变动画、点击动画、通关庆祝动画
10. 新增设置界面，可开关音效、动画、清除存档

## 界面优化
1. 圆角灯格、悬浮高亮、点击反馈
2. 状态栏美化、按钮样式统一
3. 通关/失败弹窗带动画
4. 夜间模式自动适配颜色

## 模块化要求
1. 原有模块保持不变
2. 新增独立模块：settings.py、save.py、effects.py、props.py
3. 不合并文件，不破坏原有结构
4. 所有新增功能带详细中文注释
5. 仅使用Pygame，无外部资源
6. 可直接打包EXE，中文不乱码
