# CustomTkinter Markdown 语法编辑器 V2.0（界面优化版）
基于 Python + CustomTkinter 开发桌面版 Markdown 语法编辑器 V2.0，在 V1.0 基础上全面优化界面展示与交互体验，全程现代UI组件开发，不使用外部图片资源。代码可直接运行，中文正常显示、无乱码、无BUG，支持打包EXE，适合课程作业、实训项目、日常实用工具使用。

## 界面布局（优化升级）
1. 支持窗口自由缩放、最大化、最小化，默认居中显示，界面切换无闪烁、无卡顿
2. 顶部菜单栏优化为扁平化隐藏式，鼠标悬浮展开，提升界面简洁度
3. 工具栏图标化展示，按钮增加圆角、间距、悬浮效果，操作更直观
4. 主区域左右分栏支持**拖拽调节宽度**，编辑区与预览区比例自由调整
5. 编辑区增加行号显示、语法高亮配色、光标定位高亮、选中背景突出显示
6. 预览区优化排版间距、字体、标题层级、代码块样式，接近标准 Markdown 渲染效果
7. 底部状态栏加宽加高，文字对齐更美观，增加当前主题、语法模式显示
8. 整体界面采用等高、等距、统一圆角风格，视觉更舒适专业

## 核心功能与规则
1. 基础文本操作：支持输入、删除、复制、粘贴、撤销、重做、全选
2. 完整支持 Markdown 语法：标题、加粗、斜体、列表、表格、链接、图片、代码块、引用、分割线
3. 编辑区内容实时同步预览，渲染速度更快，无延迟、无闪烁
4. 文件管理：新建、打开、保存、另存为、导出HTML，操作稳定
5. 语法高亮优化：标题、符号、代码、链接使用不同配色，辨识度更高
6. 异常处理：文件打开失败、保存失败、空内容提示，程序不崩溃
7. 保留V1.0全部稳定功能，只优化界面展示，不改动原有逻辑

## 配套功能（优化增强）
1. 浅色/深色主题一键切换，所有组件自动适配更自然的配色
2. 编辑区支持行号显示、自动换行、空格标记、语法高亮增强
3. 预览区优化字体、行距、缩进、代码高亮、表格边框、标题层级样式
4. 支持分栏拖拽缩放，适配大屏、小屏不同设备展示效果
5. 按钮、输入框、文本框增加圆角、阴影、鼠标悬浮高亮效果
6. 弹窗、提示框统一风格，与整体主题保持一致
7. 界面元素间距优化，无拥挤、无留白过多，视觉更平衡
8. 全代码中文注释，逻辑清晰，便于阅读与维护

## 代码开发规范
1. 采用面向对象方式编码，独立封装主窗口、文本编辑、实时预览、文件读写、语法解析等模块
2. 界面渲染、事件监听、文件读写、语法转换、异常处理代码完全解耦
3. 统一代码命名规则，结构层次分明，支持后续新增功能与迭代优化
4. 仅依赖Python、CustomTkinter及markdown轻量库，无冗余第三方依赖，跨设备兼容性良好
5. 采用MVC架构分层设计，视图层、控制层、逻辑层相互分离，便于后期功能拓展
6. 代码按功能模块拆分，文件不能写到一个文件中，分开写，便于维护。
