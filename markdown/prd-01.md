# CustomTkinter Markdown 语法编辑器
基于 Python + CustomTkinter 开发桌面版 Markdown 语法编辑器，全程现代UI组件开发，不使用外部图片资源。代码可直接运行，中文正常显示、无乱码、无BUG，支持打包EXE，适合课程作业、实训项目、日常实用工具使用。

## 界面布局
1. 固定窗口尺寸且支持自由缩放，窗口居中展示，界面刷新无闪烁，支持浅色/深色主题一键切换
2. 顶部设置菜单栏，包含文件、编辑、视图、格式、帮助五大分类，功能分区完整
3. 菜单栏下方配置快捷工具栏，放置新建、打开、保存、导出、清空、主题切换等功能按钮
4. 主区域采用左右分栏布局，左侧为文本编辑区，右侧为实时预览区，两区域比例协调
5. 底部设置状态栏，实时展示光标行列数、文档字数、文件路径、文档保存状态
6. 整体采用圆角、扁平化现代设计，功能分区清晰，控件排布规整，符合专业编辑器视觉风格

## 核心功能与规则
1. 基础文本操作：支持文本输入、删除、复制、粘贴、撤销、重做、全选等常规编辑功能
2. 语法渲染支持：兼容标题、加粗、斜体、有序/无序列表、表格、超链接、图片、代码块、引用、分割线等常用Markdown语法
3. 实时预览能力：编辑区内容变更后，预览区自动同步渲染效果，无需手动刷新
4. 文件管理操作：支持新建空白MD文档、打开本地MD文件、保存文档、另存为新文件
5. 格式导出功能：可将编辑内容导出为HTML文件，导出文件可直接使用浏览器打开查看
6. 语法视觉优化：编辑区实现Markdown语法高亮，区分不同语法标识
7. 异常拦截处理：针对文件打开失败、保存失败、路径无效等场景进行捕获，给出对应提示，保证程序稳定运行

## 配套功能
1. 主题适配：浅色、深色两种主题一键切换，编辑区、预览区、状态栏、工具栏自动适配配色
2. 文本排版：编辑区开启自动换行，避免长文本横向溢出
3. 数据统计：实时统计文档字符数、行数，数据同步更新
4. 快捷格式化：通过工具栏按钮，一键插入各类Markdown语法标签
5. 弹窗提示：执行新建、保存、导出、异常报错等操作时，弹出中文提示窗口
6. 界面自适应：拖动窗口大小，编辑区与预览区自动按比例缩放
7. 代码全量添加中文注释，逻辑层级清晰，方便阅读与二次修改

## 代码开发规范
1. 采用面向对象方式编码，独立封装主窗口、文本编辑、实时预览、文件读写、语法解析等模块
2. 界面渲染、事件监听、文件读写、语法转换、异常处理代码完全解耦
3. 统一代码命名规则，结构层次分明，支持后续新增功能与迭代优化
4. 仅依赖Python、CustomTkinter及markdown轻量库，无冗余第三方依赖，跨设备兼容性良好
5. 采用MVC架构分层设计，视图层、控制层、逻辑层相互分离，便于后期功能拓展
6. 代码按功能模块拆分，文件不能写到一个文件中，分开写，便于维护。
