# Pygame 创意设计搭建小游戏
使用 Python Pygame 开发一款休闲创意自由搭建设计类小游戏，纯代码绘图，无外部素材、无多余第三方依赖，代码完整可直接运行，中文显示正常无乱码，运行稳定无报错，支持打包EXE，适用于课程作业、实训项目。

## 一、界面布局要求
1. 游戏窗口居中，帧率稳定，画面流畅无闪烁。
2. 界面分区清晰：左侧素材功能工具栏、中间主设计画布、顶部状态信息栏。
3. 界面排版整洁、控件对齐、字体统一、配色简约舒适。
4. 顶部实时展示当前选中素材、操作状态、功能提示文本。

## 二、核心游戏功能
1. 内置多种几何素材：圆形、矩形、三角形、四边形、星形，可自由切换选择。
2. 鼠标选中素材后，可在中心画布自由拖拽生成、摆放图形。
3. 支持选中已绘制图形，可二次拖动调整位置。
4. 支持修改图形颜色、大小样式。
5. 支持单个图形删除、一键清空画布功能。
6. 支持将设计好的画布作品保存为本地图片。

## 三、辅助功能
1. 支持一键重置画布，重新创作。
2. 所有操作带有状态文字提示，交互完整。
3. 完善容错判断，无空操作、无闪退、无卡顿bug。
4. 支持正常退出游戏。

## 四、代码规范要求
1. 采用面向对象开发，结构分层清晰，模块化编写。
2. 拖拽逻辑、绘制逻辑、素材管理、按钮事件独立封装。
3. 代码注释详细、变量规范、逻辑严谨。
4. 纯Pygame实现，全部图形代码绘制，无需外部资源。
5. 程序稳定运行，可直接运行、可打包EXE、适合课程提交。
6. 采用mvc 架构， 分层清晰，模块化编写。每一层单独的目录。
