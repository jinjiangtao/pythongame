# CustomTkinter 停车场管理系统
基于 Python + CustomTkinter 开发桌面端停车场管理系统，纯代码实现，界面现代化美观，运行稳定无逻辑BUG，支持打包为EXE，适用于小型园区、小区、商业停车场日常车辆进出、车位管控、费用结算场景。

## 界面布局
1. 窗口固定尺寸，启动自动居中，支持浅色/深色主题一键切换，全局控件统一圆角样式。
2. 采用标签页分区布局，包含**车辆入场、车辆出场、车位监控、记录查询、费用设置、系统管理**六大模块，页面切换流畅。
3. 车辆入场/出场页：设置车牌号、车辆类型、进场时间、停放区域等录入控件，布局规整。
4. 车位监控页：可视化展示停车场分区、空闲车位/已占车位状态，不同状态用颜色区分，直观清晰。
5. 记录查询页：表格展示全部进出记录，支持分页，字段包含车牌、车型、进出时间、停放时长、应收费用。
6. 费用设置页：提供不同车型收费标准、免费时长、封顶费用配置项。
7. 底部状态栏：实时显示总车位数、剩余空位数、当前系统时间、软件运行状态。

## 核心功能与规则
1. 车辆入场登记：手动录入车牌号、选择车辆类型、停放区域，系统自动记录进场时间，同步更新车位占用状态。
2. 车辆出场结算：输入车牌号快速查询进场记录，自动计算停放时长与应付费用，确认后完成出场并释放车位。
3. 车位状态管理：实时统计总车位、已停车、空闲车位数量，车位状态实时联动更新。
4. 收费规则配置：支持区分小型车、大型车、新能源车设置不同单价，可配置免费时长、单日费用封顶规则。
5. 记录管理：所有进出记录本地持久化存储，支持单条查看、按日期/车牌号/车型筛选检索。
6. 防重复登记：同一车辆未出场时，禁止重复录入入场信息，并弹出提示。

## 配套功能
1. 数据备份与恢复功能，定期留存停车记录，防止数据丢失。
2. 关键操作增加二次确认弹窗，避免误删记录、误改收费规则。
3. 输入内容合法性校验，禁止空车牌号、非法字符录入。
4. 支持停车记录导出为本地文件，便于对账存档。
5. 操作反馈弹窗，提示登记成功、结算完成、车位已满等状态。
6. 全中文操作界面，流程简洁，上手门槛低。

## 代码开发规范
1. 采用面向对象编码，独立封装界面组件、车位逻辑、计费计算、数据读写、查询筛选、配置管理等模块。
2. 界面渲染、事件监听、业务逻辑、数据存储完全解耦，模块职责单一。
3. 代码命名规范统一，层级结构清晰，关键代码、函数、变量添加完整中文注释，方便调试与二次开发。
4. 仅依赖 Python、CustomTkinter 基础库，无额外第三方依赖，兼容性良好，可正常打包为EXE文件。
5. 按功能拆分多个文件，禁止单文件整合全部代码，便于后期维护与功能迭代。
6. 控件响应及时，大数据量记录浏览无卡顿，整体运行稳定。
