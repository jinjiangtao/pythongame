# Pygame 抓萌宠小游戏
基于 Python + Pygame 开发鼠标操控休闲小游戏，纯代码绘制图形，不使用外部图片、音频资源。代码可直接运行，运行流畅无逻辑BUG，支持打包为EXE文件，适合儿童趣味互动、专注力与反应力训练。

## 界面布局
1. 窗口固定尺寸，启动自动居中，画面划分多个随机刷新区域，布局规整
2. 游戏主区域内萌宠随机闪现、游走移动，移动轨迹灵活多变
3. 顶部实时展示当前得分、剩余时长、当前关卡，文字醒目不遮挡画面
4. 整体配色童趣柔和，角色轮廓清晰，元素大小适配鼠标点击操作，视觉干净无多余干扰

## 核心功能与规则
1. 全程鼠标操作，移动光标对准出现的萌宠单击即可完成捕捉，成功捕捉累计分数
2. 萌宠随机在不同区域出现，部分萌宠会小幅移动，停留时长各不相同后自动消失
3. 单局设置倒计时，时间耗尽游戏结束，结算最终得分
4. 萌宠刷新位置、出现时机完全随机，增加游玩不确定性

## 配套功能
1. 游戏结束后支持按键重新开局，可反复游玩
2. 配备快捷键退出游戏，操作简单易上手
3. 点击、捕捉成功均有视觉反馈，交互体验完整
4. 全代码添加中文注释，角色生成、移动逻辑、鼠标判定、计时计分逻辑说明完整清晰

## 代码开发规范
1. 采用面向对象方式编码，独立封装角色、刷新逻辑、计分计时、游戏主逻辑等模块
2. 画面渲染、鼠标事件监听、角色运动、状态切换代码相互解耦
3. 代码命名规范统一，结构层次清晰，便于阅读、调试与二次修改
4. 仅依赖 Python 与 Pygame 库，无额外第三方依赖，兼容性良好
5. 代码按功能模块拆分多个文件，禁止单文件整合，方便维护迭代
6. 界面所有提示文字使用中文，上手门槛低。
