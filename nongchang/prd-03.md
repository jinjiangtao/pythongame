# Pygame农场经营游戏 V3.0 完整版 PRD
## 项目说明
在V2.0全部功能基础上终极迭代，打造界面层级多、系统丰富、交互完善、带数据持久化的高阶农场经营游戏，MVC架构完全解耦，是高完成度课程大作业级别项目，界面复杂度、功能完整度远超普通Pygame小游戏。

## 终极新增玩法
1. 新增玩家等级经验系统，收获作物获取经验、升级解锁高级种子
2. 新增土地升级系统，可花费金币升级地块，提升作物产量、生长速度
3. 新增昼夜交替动态背景，白天黑夜画面自动切换
4. 新增本地存档系统，自动保存游戏进度，重启游戏数据不丢失
5. 新增枯萎机制，作物成熟超时未收获自动枯萎损失收益

## 界面终极复杂化
1. 多层UI嵌套：顶部状态栏、左侧功能栏、底部快捷栏、中心场景、多层弹窗、悬浮提示
2. 新增升级弹窗、等级提示弹窗、游戏结算提示UI
3. 动态画面渲染：昼夜亮度渐变、作物生长动态效果
4. 全部UI按钮hover高亮效果，交互质感拉满
5. 实时动态数据展示：经验条、升级进度、地块等级、库存详情

## 高阶完善功能
1. 完整容错机制：重复操作、超时操作、资源不足全部拦截提示
2. 多状态并存判定：地块等级、作物阶段、体力状态、等级状态互不冲突
3. 游戏暂停、继续、重置、存档、读档全套流程
4. 难度动态递增，天数越多作物生长环境越严格

## MVC完整分层
- Model：全权管理所有游戏数据、玩家属性、地块等级、存档文件、经验体系、昼夜状态
- View：全权渲染所有场景、动态背景、多层UI、弹窗、特效、进度条、悬浮信息
- Controller：全权控制所有交互逻辑、数据更新、状态切换、文件读写、游戏主流程调度
