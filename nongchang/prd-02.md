# Pygame农场经营游戏 V2.0 进阶版 PRD
## 项目说明
在V1.0基础上迭代升级，保留全部原有功能，新增多套游戏系统、多层UI面板、多样化交互逻辑，依旧采用MVC分层架构，代码模块化拓展，界面层级显著丰富，游戏完整度大幅提升。

## 核心新增玩法
1. 新增多种品类作物，不同作物生长时长、售价差异化
2. 新增玩家体力系统，种植、收获消耗体力，自动定时恢复
3. 新增背包系统，收获作物自动存入背包，可查看作物库存
4. 新增商店系统，弹窗打开，支持售卖背包作物、购买种子

## 界面升级
1. 新增左侧固定功能栏：背包按钮、商店按钮、刷新按钮
2. 新增底部快捷操作栏：种子选择面板
3. 新增弹窗UI：商店弹窗、背包弹窗，支持开关切换
4. 顶部状态栏新增：体力值、作物数量实时展示
5. 新增鼠标悬停提示，显示地块作物生长进度

## 新增功能
1. 种子选择切换，不同地块可种植不同作物
2. 体力不足禁止操作，完善容错交互
3. 作物库存管理、批量售卖机制
4. 难度与时间适配，天数递增作物生长微调

## MVC分层拓展
- Model：新增体力数据、背包数据、商店商品数据、种子数据
- View：新增侧边栏、底部栏、弹窗界面、悬浮提示渲染
- Controller：新增按钮点击事件、弹窗开关、背包商店交互逻辑
