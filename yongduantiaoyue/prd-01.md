# Pygame 云端平台跳跃休闲游戏
基于 Python + Pygame 开发云端平台跳跃类小游戏，全程纯代码绘制图形与界面，不使用外部图片、音频资源。代码可直接运行，中文展示正常，运行流畅无卡顿、无逻辑BUG，支持打包为EXE文件，适配课程作业与实训项目使用。

## 界面布局
1. 固定游戏窗口尺寸，保持稳定帧率，画面刷新无闪烁
2. 顶部状态栏：实时展示得分、当前关卡、游戏状态提示
3. 中间主区域：蓝天白云背景 + 随机生成云朵平台，作为核心游玩区域
4. 整体配色清新柔和，界面元素排布规整，视觉体验舒适

## 核心玩法与规则
1. 键盘左右方向键控制角色左右移动，空格键实现跳跃，支持二段跳
2. 云朵平台整体匀速向下滚动，角色站立在平台上会跟随平台同步移动
3. 角色踩中全新云朵平台累计分数，分数实时更新展示
4. 角色掉落出屏幕底部判定游戏结束，展示最终得分并提供重新开始选项
5. 关卡递进机制：关卡越高，平台下移速度越快、平台分布间距越大，难度逐步提升
6. 云朵平台随机生成位置与尺寸，保证每局游玩体验不同

## 配套功能
1. 数据展示：侧边栏实时显示当前得分、当前关卡、简易操作说明
2. 状态提示：对局中、游戏结束、重新开局均搭配中文文字提示
3. 基础操作：支持游戏重新开局、退出游戏功能
4. 逻辑优化：完善边界检测、落地判定、跳跃逻辑，避免角色穿墙、卡死、判定异常等问题
5. 开局引导：游戏启动后显示玩法说明界面，帮助新手快速上手
6. 代码注释：所有游戏元素、运动逻辑、碰撞判定、事件处理均添加中文注释
7. 字体适配：中文正常渲染无乱码，文字清晰易读

## 代码开发规范
1. 采用面向对象思想编码，拆分角色、平台、游戏主逻辑等独立模块
2. 事件监听、角色运动、物理碰撞、计分规则、画面绘制代码相互解耦
3. 代码命名规范，关键逻辑补充中文注释，结构清晰，便于阅读与二次修改
4. 仅依赖 Pygame 库，无额外第三方依赖，兼容性强
5. 整体采用 MVC 架构分层设计，模块独立，方便后期维护与功能扩展
