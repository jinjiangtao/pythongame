使用 Python + Pygame 实现**单机单人版**俯视2D经典坦克大战游戏，**禁止所有代码写在同一文件**，严格模块化分包开发，使用 unittest 编写独立测试文件，结构清晰、低耦合。

特别注意中文展示问题，确保游戏窗口中所有文字都能正常显示。

目录结构：
- main.py 程序入口与主循环
- src/
  - tank.py 坦克类（玩家、敌方AI）
  - bullet.py 子弹类
  - map.py 地图、墙体、基地
  - game.py 游戏核心逻辑
  - config.py 全局配置
- tests/
  - test_tank.py
  - test_bullet.py
  - test_map.py

游戏规则（纯单机）：
1. 仅玩家单人操作，无双人模式；
2. 玩家通过 WASD 控制坦克上下左右移动，空格键发射子弹；
3. 敌方AI坦克自动随机巡逻、主动射击，不能穿墙；
4. 地图包含可摧毁砖墙、不可摧毁铁墙、草丛遮挡；
5. 我方基地被敌方子弹击中 → 游戏失败；
6. 消灭当前地图所有敌方坦克 → 通关，可进入下一关；
7. 玩家坦克有血量，中弹扣血，血量为0游戏结束；
8. 子弹飞出边界自动销毁，坦克不可超出游戏窗口边界；

要求：类职责单一，测试用例可独立运行，注释规范，可扩展多关卡。
