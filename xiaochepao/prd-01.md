# Pygame 小车跑酷力学小游戏
基于 Python + Pygame 开发横版小车跑酷游戏，纯代码绘制图形，不使用外部图片、音频资源。代码可直接运行，运行流畅无逻辑BUG，支持打包为EXE文件，适用于力学知识启蒙、趣味互动练习。

## 界面布局
1. 窗口固定尺寸，启动自动居中，画面分层设计地面与背景，视觉层次清晰
2. 画面整体从左至右滚动，小车固定相对位置，障碍物持续从右侧出现
3. 顶部区域实时展示当前分数、操作提示，字体醒目不遮挡游戏画面
4. 整体配色简洁护眼，图形轮廓分明，元素大小适中，观感舒适无杂乱内容

## 核心功能与规则
1. 小车保持持续向前的运动状态，模拟物体惯性运动效果
2. 按下空格键控制小车执行跳跃动作，跳跃过程模拟真实重力，上升、下落轨迹自然
3. 游戏全程自动生成障碍物，障碍物随机间隔出现，实现无限闯关玩法
4. 小车与障碍物发生碰撞则判定游戏结束，跑动距离累计对应得分
5. 得分随游戏运行持续增加，跑动越久分数越高

## 配套功能
1. 支持按键重新开始游戏，一局结束后可快速再次游玩
2. 支持快捷键退出游戏，操作简单易懂
3. 运动轨迹、重力下落效果真实还原基础力学现象
4. 全代码添加中文注释，运动逻辑、碰撞判断、事件监听说明完整清晰

## 代码开发规范
1. 采用面向对象方式编码，独立封装小车、障碍物、游戏主逻辑等模块
2. 画面渲染、按键监听、物理运动、碰撞检测代码相互解耦
3. 代码命名规范统一，结构层次清晰，便于阅读、调试与二次修改
4. 仅依赖 Python 与 Pygame 库，无额外第三方依赖，兼容性良好
5. 代码按功能模块拆分，文件不能写到一个文件中，分开写，便于维护。
6. 界面与提示文字均使用中文，上手门槛低。
7. 物理参数调试合理，运动效果自然，贴合基础力学演示需求。
8. 代码不能写到一个文件中，按模块 拆开文件，方便后面维护和扩展。


在现在功能的基础上，完成下面的迭代。
1. 动态变速系统：游戏分数越高，场景滚动速度持续递增，后期节奏极速变快，难度平滑升级。
2. 多形态复合型障碍物：随机生成矮障碍、高墙障碍、空中悬浮障碍、连续连环障碍，需要精准判断跳跃时机。
3. 二段跳进阶操作：支持空中二段跳跃，既增加操作上限，又能锻炼节奏把控，区别普通新手版本。
4. 随机增益道具系统：随机刷新减速、无敌、双倍分数三种道具，增加游戏策略性与可玩性。
5. 精准碰撞判定优化：采用像素级精细碰撞，区分擦边碰撞、完全撞击，不再是简单矩形判定，提升游戏专业性。
